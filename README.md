Game hứng táo chơi bằng cách nghiêng điện thoại

Đây là project để ôn lại kiến thức

Những thứ có trong dự án này
- cấu trúc : mvvm
- ui : compose
- roomdatabase
- share datastore
- firebase (firestore database, crashlytics, analytics)
- koin for dependency injection
- coil
- material 3
- navigation 3

Logic:
- Màn game details:
  - Vẽ các vật thể trên canvas 
  - Lấy dữ liệu từ SensorEventListener để điều khiển giỏ
  - Tạo ngẫu nhiên các vật thể (bom, coin, táo) sau đó tính toán va chạm và xác định điểm, máu, tiền
- Màn store
  - Lấy data từ firebase
  - Nếu người dùng mua sẽ lưu item vào bộ nhớ điện thoại
